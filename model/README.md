# MBTI 번역기 - 모델

MBTI 성격 유형별 텍스트 스타일 변환 모델

## 디렉토리 구조

```
model/
├── data/              # 데이터셋 저장
├── notebooks/         # 실험 및 분석용 노트북
├── src/              # 소스 코드
│   ├── preprocessing/ # 데이터 전처리
│   ├── training/     # 모델 학습
│   └── inference/    # 모델 추론
└── requirements.txt  # 필요한 패키지
```

## 설치 방법

```bash
pip install -r requirements.txt
```
